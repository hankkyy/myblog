OpenAI 宫斗续集：Sam Altman 回归一年后 OpenAI 变成什么样了？

2024 年底 Sam Altman 回归 OpenAI 满一周年。回头看过去一年，OpenAI 变化太大了。

重组

OpenAI 宣布转型为营利性公司（PBC，Public Benefit Corporation）。这意味着： - 不再受非营利董事会限制 - 可以正常融资和上市 - 但也引发了很多争议——马斯克起诉说这背叛了 OpenAI 的初衷

人才流动

过去一年离开 OpenAI 的核心人员： - Ilya Sutskever（联合创始人，去了 Safe Superintelligence） - Mira Murati（前 CTO） - 多位安全团队成员

Altman 的说法是「这是正常的人才流动」，但外界解读为安全派和商业派的路线之争。

产品

  • ChatGPT 月活从 1 亿增长到 3 亿
  • GPT-4 Turbo 和 GPT-4o 发布
  • Sora 视频生成模型推出
  • 跟苹果的合作（Apple Intelligence 集成 ChatGPT）

OpenAI 不再是那个「非营利研究实验室」了，它正在变成另一个科技巨头。
