Cursor 估值突破 100 亿美元：AI 编程工具重新定义了「IDE」

Cursor 的估值到了 100 亿美元。一个「代码编辑器」值这么多？

做了什么不一样

Cursor 不是给传统 IDE 加个 AI 插件，而是从零设计的 AI 原生编辑器： - AI 可以读懂整个代码库的上下文 - 一个指令就能跨多个文件重构 - 自动生成测试、文档、PR 描述 - 「Agent 模式」可以自主完成一个完整功能

跟 GitHub Copilot 的区别

  • Copilot 是「IDE 里的 AI 助手」
  • Cursor 是「AI 助手就是 IDE」
  • Cursor 的上下文理解深度远超 Copilot
  • 但 Copilot 有 GitHub 生态的优势（PR、Issues、Actions 深度集成）

开发者生态变化

  • 初级开发者的效率提升最大——AI 帮你写大部分代码
  • 资深开发者的角色从「写代码」变成「审代码 + 定方向」
  • 「10x 工程师」的定义变了：不是打字快，而是能高效跟 AI 协作

争议

有人担心过度依赖 AI 会导致开发者退化——就像用计算器久了不会心算。但更多人觉得这就是工具进化，不必焦虑。

不管怎样，不会用 AI 编程工具的人，效率确实会落后。
