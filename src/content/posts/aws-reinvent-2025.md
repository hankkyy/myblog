AWS re:Invent 2025 的重点全是 AI。

## Trainium 3 芯片

Amazon 自研的第三代 AI 训练芯片：
- FP8 算力：4 PFLOPS（对标 H100 的 2 PFLOPS）
- 能效比：比 H100 高 40%
- 最大集群：10 万张卡
- 定位：不跟英伟达比峰值性能，主打性价比

## Amazon Nova

Nova 是 AWS 自己的多模态大模型：
- 支持文本、图像、视频理解
- 跟 AWS 服务（S3、DynamoDB、Lambda）深度集成
- 定位是「实用型 AI」而不是「最强 AI」

## Amazon Q 2.0

面向开发者的 AI 编码助手，对标 GitHub Copilot：
- 支持代码生成、Review、测试
- 跟 AWS 控制台深度整合
- 定价比 Copilot 便宜 30%

## 战略意义

AWS 的 AI 策略很清晰：不追求模型最强，而是做「最方便用的 AI 云计算」。开发者不需要离开 AWS 生态就能用 AI——从训练到推理到应用，全链条覆盖。

这可能是对抗微软（Azure + OpenAI）最有效的策略。
